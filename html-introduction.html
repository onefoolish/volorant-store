什么是 HTML？

HTML（HyperText Markup Language，超文本标记语言）是构建网页的标准语言。 它使用一系列标签（tags）来描述网页的结构和内容，例如标题、段落、图像、链接等。 浏览器通过解析 HTML 代码，将这些元素渲染成用户可见的页面。

HTML 的基本结构

每个 HTML 文档都有一个标准结构： 

<!DOCTYPE html>
<html>
  <head>
    <title>页面标题</title>
  </head>
  <body>
    <h1>这是一个主标题</h1>
    <p>这是一个段落。</p>
  </body>
</html>

常见标签示例

  • <h1> 到 <h6>：表示不同级别的标题
  • <p>：定义段落
  • <a href="url">：创建超链接
  • <img src="image.jpg" alt="描述" />：插入图片
  • <ul>, <ol>, <li>：创建列表

HTML 与 Web 开发

HTML 通常与 CSS 和 JavaScript 一起使用： CSS 负责样式设计（颜色、布局、字体等）， JavaScript 负责交互行为（点击、动画、数据加载等）。 三者共同构成了现代网页开发的基础。
